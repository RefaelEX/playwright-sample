## Sample playwright project 
Automation tests implementation with playwright

#### Build: 
  * Run `yarn install`

#### Install playwright browsers: 
  * Run `yarn playwright install chromium`

####  Start test page:
  * Run `yarn testPage:start`

####  Run ui tests:
  * Run `yarn test:ui`

####  Run api tests:
  * Run `yarn test:api`

#### Generate Report :
  * Run `yarn report`
